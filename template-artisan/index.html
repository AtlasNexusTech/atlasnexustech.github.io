Questions fréquentes

Comment ça marche après l'achat ?

Vous recevez un ZIP par email. Vous modifiez le fichier config.json avec vos infos (nom, téléphone, services…), vous glissez-déposez sur GitHub Pages, et votre site est en ligne. 5 minutes montre en main.

J'ai besoin de compétences techniques ?

Zéro. Le config.json est un fichier texte simple. Vous remplacez les infos d'exemple par les vôtres. Le guide explique tout pas à pas avec captures d'écran.

C'est quoi GitHub Pages ? C'est payant ?

GitHub Pages est un hébergement gratuit et illimité. Vous créez un compte gratuit en 2 minutes. Votre site sera en votrenom.github.io ou sur votre propre nom de domaine.

Je peux utiliser mon propre nom de domaine ?

Oui. Le guide explique comment connecter mon-artisan.fr à votre site GitHub Pages. C'est gratuit et ça prend 5 minutes.

Et si je veux un site différent des démos ?

Le secteur "générique" est entièrement personnalisable. Vous choisissez vos couleurs, vos icônes, vos services. Ou contactez-nous pour un site sur mesure.
