Ce qui rend Atlas Desk différent

Pas de cloud propriétaire, pas de compte, pas de limite de temps. Du P2P pur sur WebRTC.

🔐

Chiffrement bout-en-bout

WebRTC impose DTLS-SRTP. Même le signaling ne voit pas vos données.

🖥️

Client web zéro installation

Un navigateur suffit pour contrôler un PC distant. Pas de .exe, pas de plugin.

⚡

Agent léger en Go

Binaire statique de ~15 MB. Démarre en <1s. 15 FPS en streaming.

📋

Clipboard synchronisé

Copiez sur votre PC, collez sur le remote. Fonctionne dans les deux sens.

📁

Transfert de fichiers

Envoyez et recevez des fichiers directement via le canal P2P. Sans intermédiaire.

🔑

Auth par mot de passe

Challenge SHA-256. Votre PC, votre mot de passe. Pas de compte cloud.

Comment ça marche

Trois composants. Aucun abonnement. Le signaling est auto-hébergeable.

1

Agent sur votre PC

Binaire Go. Capture l'écran, stream via WebRTC.

2

Signaling server

Échange les offres SDP. Ne voit jamais vos données.

3

Client web

Votre navigateur. Contrôle clavier/souris en direct.

# Démarrez l'agent sur le PC à contrôler $ ./atlas-desk-agent -password monmotdepasse # Ouvrez le client web et connectez-vous avec l'ID 9 chiffres → https://atlasnexus.tech/atlas-desk/client
