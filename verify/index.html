Vérification cryptographique on-chain. Enregistrement ERC-8004, signature, consultation du registre. Aucune donnée personnelle collectée.

Comment ça fonctionne

1️⃣

Enregistrement

Chaque agent reçoit un NFT d'identité unique (ERC-8004) sur la blockchain Celo.

2️⃣

Signature

L'agent signe un message avec sa clé privée pour prouver qu'il détient l'identité.

3️⃣

Vérification

N'importe qui peut vérifier instantanément l'identité d'un agent via le registre on-chain.

🔧 Outil en maintenance

L'interface interactive de connexion wallet est temporairement désactivée pour maintenance de sécurité. L'outil reste pleinement fonctionnel on-chain.
